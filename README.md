## 🚀 Getting Started

### Prerequisites
- Node.js (v14+)
- TMDB API Key (Free)

### Installation

```bash
git clone https://github.com/ThereIsSomething/
cd movie-wishlist
npm install
```

### Setup

Create a `.env` file in the root directory and add:

```env
VITE_TMDB_API_KEY=your_tmdb_key
```

### Run

```bash
npm run dev
```
